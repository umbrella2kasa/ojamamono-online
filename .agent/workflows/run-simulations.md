# シミュレーション実行ガイド

サーバー側のゲームロジック（カード配置、AIの挙動、勝敗判定など）を検証するための手順です。

## 実行手順

1. サーバーディレクトリに移動するか、ルートから以下のコマンドを実行します。
// turbo
2. `npm run test:sim` (未設定の場合は `npx ts-node src/simulation.ts` など) を実行してシミュレーションを開始します。

## ログの確認

シミュレーションが完了すると、`server/` ディレクトリに以下のファイルが生成されることがあります：

- `debug_trace_vX.txt`: 詳細な実行ログ
- `simulation_report.md`: 集計結果のレポート

## 注意事項

- シミュレーションは CPU リソースを消費するため、実行回数（Iteration）に注意してください。
- ロジックを変更した後は、`shared` と `server` をビルドし直す必要がある場合があります。
